# Surface
毛玻璃容器，胶囊或面板两种形态。

## Props
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| variant | `'capsule' \| 'panel'` | `'capsule'` | 形态 |
| children | `ReactNode` | - | 内容 |
| className | `string` | `''` | 额外类名 |

## Usage
```tsx
import Surface from './Surface';
<Surface variant="panel">内容</Surface>
```
