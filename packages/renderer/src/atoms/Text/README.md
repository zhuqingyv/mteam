# Text
文本排版原子，四种变体。

## Props
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| variant | `'title' \| 'subtitle' \| 'caption' \| 'badge'` | `'caption'` | 文本样式 |
| children | `ReactNode` | - | 内容 |

## Usage
```tsx
import Text from './Text';
<Text variant="title">M-TEAM</Text>
```
