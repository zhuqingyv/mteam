# StatusDot
状态圆点，绿/黄/灰三色。

## Props
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| status | `'online' \| 'busy' \| 'offline'` | `'online'` | 状态颜色 |
| size | `'sm' \| 'md' \| 'lg'` | `'md'` | 尺寸 |

## Usage
```tsx
import StatusDot from './StatusDot';
<StatusDot status="online" size="md" />
```
